ppm-commons
===========

PPM Commons Library.

It requires JDK 11+ if you want JDK 8 compatibility go for version 2.1.5 or below.

To use ppm-commons add to your pom file:
```
<dependency>
  <groupId>br.com.ppm</groupId>
  <artifactId>ppm-commons</artifactId>
  <version>2.2.2</version>
</dependency>
```
