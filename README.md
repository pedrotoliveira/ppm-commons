ppm-commons
===========

PPM Commons Library.

Requires JDK 8+

[![Maven Central](https://img.shields.io/maven-central/v/org.apache.maven/apache-maven.svg)]()

To use ppm-commons add to your pom file:
```
<dependency>
  <groupId>br.com.ppm</groupId>
  <artifactId>ppm-commons</artifactId>
  <version>1.2</version>
</dependency>
```
